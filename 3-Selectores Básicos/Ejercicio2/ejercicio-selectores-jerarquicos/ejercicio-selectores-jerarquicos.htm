  • 2. Los elementos de tipo "div" que tienen asignadas la clase "bodyPreviewAlert" y la clase "sbPreviewAl". (div.bodyPreviewAlert.footPreviewAlert)

  • 6. Los elementos de tipo "li" hijos de elementos "div" con clase "notificacionesDisplay". (div.notificacionesDisplay > li)
  • 7. Los elementos de tipo "li" con clase "messageContent" hijos de elementos "ul" con clase "mensajesBox". (ul.mensajesBox > li.messageContent)
  • 8. Los elementos de tipo "a" con clase "readMore" hijos de un "li" que es el hermano siguiente de un "li" con la clase "messageContent". (li.loadAlerts + li > a.readMore)
